### Моделирование случайных событий (GUI)

**Часть 1:**
Приложение «Скажи “да” или “нет”».

**Часть 2:**
Приложение «Шар предсказаний» (Magic 8-Ball).

---

#### Описание

Данный лабораторный проект имитирует случайные события с помощью собственного генератора псевдослучайных чисел. Приложение разработано с использованием HTML, CSS, JavaScript на стороне клиента и Golang на стороне сервера и включает два независимых модуля: «Да/Нет» и «Предсказание шара 8».

Проект основан на мультипликативном конгруэнтном генераторе (MCG), который генерирует последовательность псевдослучайных чисел в диапазоне [0,1). Генератор инициализируется начальным значением текущей метки времени Unix, которое может быть задано пользователем через графический интерфейс. Это обеспечивает воспроизводимость результатов моделирования.
