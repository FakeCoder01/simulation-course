## 2. Теоретическая справка

Основная идея стохастического моделирования заключается в установлении закономерностей с использованием статистических свойств случайных объектов.

**Базовый датчик** — это алгоритм или программный компонент, который генерирует реализации непрерывной случайной величины $\alpha$, равномерно распределенной в интервале $[0; 1]$. Большинство датчиков других случайных объектов используют базовый датчик путем математического преобразования полученной величины $\alpha$ в нужный случайный объект $X$.

К базовым датчикам предъявляются следующие требования:

- **Математические:** равномерное распределение, отсутствие корреляции, апериодичность, воспроизводимость.
- **Вычислительные:** быстрота.
- **Общие:** портируемость.

В данной работе в качестве псевдослучайного датчика реализован линейный конгруэнтный метод (основанный на мультипликативном конгруэнтном генераторе). Последовательность псевдослучайных чисел формируется по формуле:
$$X_{n+1} = (a \cdot X_n + c) \mod m$$
Для получения значения в интервале $[0; 1)$ результат делится на $m$.

**Теоретические характеристики равномерного распределения на отрезке $[0; 1]$:**

- Теоретическое среднее (математическое ожидание): $E[X] = \frac{0 + 1}{2} = 0.5$
- Теоретическая дисперсия: $V[X] = \frac{(1 - 0)^2}{12} \approx 0.083333$

Программа реализована на языке Rust.Для воспроизводимости результатов используется начальное значение (зерно или seed).
